This project uses data from Instacart, a popular grocery delivery platform similar to Uber Eats or Door Dash. The original dataset was released in 2017 for a Kaggle competition, and here we work with a reduced version that also includes intentional missing values and duplicates to simulate more realistic, complex data.

Goal: Clean and analyze the dataset to uncover valuable insights about customer purchasing habits and trends. Results and explanations were documented in a Jupyter Notebook using Markdown cells.

Technologies: Python 3.x, Pandas, Matplotlib
