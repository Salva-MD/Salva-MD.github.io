As a data analyst for Megaline, a telecom company with two prepaid plans (Surf and Ultimate), I analyzed which plan generates higher revenue to help optimize the marketing budget.

Using a dataset of 500 Megaline clients, the analysis included:

  • Customer information (location, plan subscribed)
  • Number of calls made
  • Text messages sent
  • Mobile data usage
  • Revenue generated per customer

An statistical analysis was conducted to compare revenues and determine which plan contributes more on average.

Objective: Analyze customer behavior by plan, compare generated revenue, and apply statistical tests to verify if differences are significant.

Instructions for execution: Clone the repository and install dependencies with pip install -r requirements.txt.

Python, Pandas, Matplotlib (pyplot), Numpy, Seaborn, Scipy (stats), Math
